# MainCrafts Landing Page

This project is a modern and responsive landing page created as part of a Full Stack Web Development Internship Task.

## 🚀 Features
- Responsive design (mobile, tablet, desktop)
- Navigation bar with dropdown menu
- Hero section with call-to-action button
- Feature cards with icons
- Clean and simple footer
- Modern UI using CSS (Flexbox, hover effects, gradients)

## 🛠️ Technologies Used
- HTML5
- CSS3
- Google Fonts
- Font Awesome
